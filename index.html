🚀 วิธี Upload ขึ้น GitHub Pages
1
สร้าง Repository

ไปที่ github.com → New repository ตั้งชื่อ เช่น hotel-finance-2026 และเลือก Public

2
Upload ไฟล์ทั้งหมด

คลิก "Add file → Upload files" แล้วลาก ไฟล์ทั้ง 5 ไฟล์ (index.html + 4 dashboards) ขึ้นพร้อมกัน

3
เปิด GitHub Pages

ไปที่ Settings → Pages → Source เลือก main branch / root แล้วกด Save

4
รอ 1-2 นาที

GitHub จะแสดง URL เช่น https://username.github.io/hotel-finance-2026
